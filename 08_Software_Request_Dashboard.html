Import Log Data

Paste JSON from the Submission Log's localStorage, or upload the CSV exported via the ⬇ Export CSV button on the log page. Data is saved locally in this browser.

Export CSV from the Submission Log page, then upload it here

Option A — Same browser (auto-sync):
If you open both log.html and this dashboard in the same browser, they share the same localStorage key (moss_software_requests_log) and data syncs automatically. No import needed — just click ↻ Refresh.

Option B — Export CSV from the Log:
1. Open log.html in your browser.
2. Click ⬇ Export CSV.
3. Return here, click ⬆ Import Log Data, switch to the CSV tab, and upload the file.

Option C — Copy JSON from localStorage:
1. Open log.html.
2. Open DevTools → Application → Local Storage → this site's origin.
3. Copy the value of moss_software_requests_log.
4. Paste it in the JSON tab here and click Import.
